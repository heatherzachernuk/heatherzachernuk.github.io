About Petulant Tic-Tac-Toe

There's nothing fancy going on here, except when your AI opponent loses.

Everyone starting out in web-development builds a game of tic-tac-toe. I wanted to explore how it feels when a computer player has a petty human-like response to losing a silly game.

Technical details

The exciting action is in the javascript class "ScribbleCanvas". When invoked by the ending of a game, ScribbleCanvas generates an HTML canvas over the game board.

The location of each move by the computer has been stored in an array. The function "loadPoints" takes each location in turn, finds the middle of the square and then draws a line from each square to the next.

Creating the hand-drawn look

The animated jitter by the petulant computer player is created by drawing a line using the canvas properties setLineDash and lineDashOffset.

Each section of the line is divided into segments. The end point of each segment is randomly offset to create a hand-drawn look.
