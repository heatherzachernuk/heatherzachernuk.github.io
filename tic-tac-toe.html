About Petulant Tic-Tac-Toe

There's nothing fancy going on here, except when your AI opponent loses.

Everyone starting out in web-development builds a game of tic-tac-toe. I wanted to explore what it feels like when a computer player acts like a petty baby.

Technical details

The exciting action is in the javascript class "ScribbleCanvas". When invoked by the ending of a game, ScribbleCanvas generates an HTML canvas over the game board.

The location of each move by the computer player has been stored in an array "compLocations". The function "loadPoints" takes each location in turn, finds the middle of the square and then draws a line from each square to the next. The function "update" adds some styling to how the line is drawn, to achieve a jittery, petulant, hand-drawn look.
