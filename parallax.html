Parallax image experiment

I was inspired by the effect of the beautiful parallax graphic for the homepage of the game ‘Firewatch’, and decided to experiment with the concept using HTML/CSS.

How it works

The image consists of seven PNG layers translated along points in the z-axis, using the CSS translateZ function. When the user scrolls, each layer moves at a slightly different speed, creating the illusion of depth.

Subject

I constructed a cityscape in order to explore ideas of scale. I hoped to create a sense of discovery and zooming in as the user scrolled, starting with a view of distant skyscrapers and gradually uncovering a candid inner-city slum scene.
